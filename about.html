Game Design student • Building toward Send It Games

I’m Deon, a Game Design student focused on building the skills and habits needed to launch my own studio, Send It Games. I’m interested in projects that reward mastery and experimentation, where players can feel themselves improve instead of being dragged along by cutscenes and pity mechanics.

What this site is

This GitHub Pages site is my running newsletter for CAH 205. I’m using it to track applied AI tools and updates that creators actually use, especially tools that improve workflows instead of just creating flashy demos. If the update saves time, improves iteration speed, or makes output more controllable, it’s worth writing about.

What I’m building toward

  • Send It Games: an indie studio with an emphasis on skill-based, high-engagement game design.
  • Better production habits: consistent iteration, clear scope control, and shipping work in public.
  • Applied AI fluency: using AI as a tool for speed and structure, not a substitute for judgment.
